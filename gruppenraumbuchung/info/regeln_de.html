1. Reservieren können nur Angehörige der Universität Trier.
  2. Eine Vormerkung verfällt, wenn sie nicht spätestens am Tag vor dem Termin in eine Reservierung umgewandelt worden ist.
  3. Für den laufenden Tag können keine Online-Reservierungen vorgenommen werden.
  4. Jeder Benutzer kann pro Kalenderwoche drei Reservierungen vornehmen.
  5. Jeder Benutzer kann pro Tag eine Reservierung vornehmen.
  6. Ein Gruppenarbeitsraum kann für höchstens drei Stunden reserviert werden.
  7. Gruppenarbeitsräume können höchstens sieben Tage im voraus gebucht werden.
